## template

Get familiar with template functions and classes, as well as standard tempalte library (STL).

### Run

* Linux
```bash
mkdir build
cd build
cmake ../
make
./test_mbuf
```

* Windows with Powershell
```bash
mkdir build
cd build
cmake ../ -G "MinGW Makefiles"
mingw32-make.exe
test_mbuf.exe
```
